## What I do

I design and ship **platform automation systems** that turn manual infrastructure operations into repeatable, code-driven pipelines — from Day-0 provisioning through Day-N lifecycle management.

Most of my career was spent at **Hewlett Packard Enterprise**, where I progressed from Specialist → Senior Specialist → Expert, leading the architecture and delivery of:

- **VMaaS / VMCaaS** — VMware-as-a-Service automation platform built from scratch. Schema-driven Service Definition engine, Terraform orchestration, NSX-T, Fiber Channel, Site HA, and ~100% deployment automation coverage across enterprise customer builds
- **AI Core Engine (2018)** — An NLP/NLU-based conversational system for enterprise infrastructure queries, built before LLMs were mainstream. Won HPE's Parasparam Innovation Award
- **STINT** — HPE Synergy infrastructure deployment automation; parallel workflow design, configuration discovery via REST APIs

I think in systems. I write the code. I mentor the team.
