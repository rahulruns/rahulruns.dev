About
The longer version

I build the automation systems that make enterprise infrastructure deployable at scale — from zero to fully orchestrated, Day-0 through Day-N.

Over 21 years, I progressed from shell scripting on AIX servers to architecting VMware-as-a-Service platforms at HPE, reaching Expert level. I've learned that the hardest part of infrastructure automation isn't the code — it's designing systems that are reentrant, schema-driven, and robust enough for real production.

I built an AI Core Engine back in 2018 — a natural language query system for enterprise infrastructure — long before LLMs were mainstream. That same curiosity now drives my work with LangGraph and Text-to-SQL systems.

Currently building in public: open-source infrastructure tools and AI-assisted platform engineering systems that distil what I learned at enterprise scale.
